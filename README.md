# Discord Bot for SOL/USDT RSI
Discord bot that's fetchs candlestick chart data by BYBIT's API, calculates it's RSI(14) and alerts when it's value is over 70 or below 30
<br/>
<br/>
## Requierments
<ul>
<li>discord [2.3.2]</li>
<li>requests [2.32.3]</li>
<li>pandas [2.2.2]</li>
</ul>
